# 루틴 타이머

로컬 전용 루틴 순차 타이머 앱 (Expo / React Native).

## 기능

- 루틴·단계 CRUD
- 카드 드래그로 단계 순서 변경
- 순차 타이머 (일시정지 / 스킵 / 이전 / 종료)
- 사운드·햅틱 설정
- AsyncStorage 로컬 저장
- 백그라운드 단계 전환 알림

## 실행

```bash
pnpm start
```

Expo Go 앱으로 QR을 스캔하거나 `a`(Android) / `i`(iOS) / `w`(web)를 사용하세요.
