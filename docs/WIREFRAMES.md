# 화면 와이어프레임 (MVP)

## 홈 — 루틴 목록

```
┌─────────────────────────┐
│  내 루틴          ⚙     │
│                         │
│  ┌───────────────────┐  │
│  │ 아침 루틴          │  │
│  │ 18분 · 3단계   ▶  │  │
│  └───────────────────┘  │
│  ┌───────────────────┐  │
│  │ 공부 포모도로      │  │
│  │ 30분 · 2단계   ▶  │  │
│  └───────────────────┘  │
│                         │
│              ┌───┐      │
│              │ + │      │
│              └───┘      │
└─────────────────────────┘
```

- 카드 탭 → 상세 편집
- ▶ 탭 → 바로 실행
- + → 새 루틴 생성 후 상세로 이동

## 루틴 상세 — 편집 + 순서

```
┌─────────────────────────┐
│ ←  아침 루틴        ⋯   │
│     총 18분 · 3단계     │
│                         │
│  ≡  스트레칭      5:00  │
│  ≡  세수          3:00  │
│  ≡  명상         10:00  │
│                         │
│  [ + 단계 추가 ]        │
│                         │
│  ┌───────────────────┐  │
│  │      시  작       │  │
│  └───────────────────┘  │
└─────────────────────────┘
```

- ≡ 드래그로 순서 변경
- 카드 탭 → 단계 편집 시트
- ⋯ → 복제 / 삭제

## 단계 편집 시트

```
┌─────────────────────────┐
│      단계 편집          │
│  제목 [___________]     │
│  분  [ -  5  + ]        │
│  초  [ -  0  + ]        │
│  [삭제]        [저장]   │
└─────────────────────────┘
```
